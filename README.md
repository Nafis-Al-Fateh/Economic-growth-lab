# Economic Growth Lab – Solow Growth Model Simulator

An interactive web application that demonstrates the dynamics of the **Solow Growth Model**.  
Users can adjust key macroeconomic parameters and observe how they influence long-run economic growth and steady-state outcomes.

The application is built using **Python** and **Streamlit** and is designed as an educational and research visualization tool for students of macroeconomics.

## Project Overview

The Solow Growth Model explains long-run economic growth through:

- Capital accumulation
- Population growth
- Technological progress

This interactive simulator allows users to explore how changes in these parameters affect:

- Capital per worker
- Output per worker
- Long-run steady-state equilibrium

---

## Features

### Interactive Parameter Control
Users can adjust key economic parameters:

- Savings Rate (s)
- Capital Share (α)
- Depreciation Rate (δ)
- Population Growth (n)
- Technological Growth (g)
- Initial Capital
- Simulation Time Period

### Dynamic Simulation
The application simulates the evolution of:

- Capital per worker
- Output per worker

over time.

### Automatic Economic Interpretation
The system provides real-time economic interpretations based on user-selected parameters.

### Steady-State Calculator
The model automatically computes:

- Steady-state capital
- Steady-state output
